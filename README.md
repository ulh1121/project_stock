# project_stock

### bootstrap 템플릿을 이용하여 주식 관련 정보를 다양하게 나타내고, 다음 장날의 주가 정보를 제공하는 웹페이지 생성

<img width="100%" src="https://user-images.githubusercontent.com/67838477/162278544-2a6e5979-e3ee-4d09-afff-ec17165b6201.PNG">

- 현재 주가와 함께 국내 지수, 미국 지수 표기, 설정 기간만큼의 주가 추이 차트로 구현
- ARIMA모델을 활용해 다음 장날 주가 예측. 예측에는 한달분량의 주가 데이터를 이용했고, 모델 update를 통한 예측 방식 사용

<img width="100%" src="https://user-images.githubusercontent.com/67838477/162279628-ca3b1dc3-ef40-4596-a636-483fb1e90489.PNG">

- 연관 기업과의 지표 비교, 매출과 당기순이익, 네이버금융 페이지의 투자 컨센서스를 통한 매수매도 의견 시각화

<img width="100%" src="https://user-images.githubusercontent.com/67838477/162282367-8738e34a-2492-4dd8-9313-8e28af550b0b.PNG">

- 재무제표, 최신 뉴스 크롤링
- 주식 관련 주요 지표의 정도를 날씨로 표현해 주식 예보 카테고리 생성
